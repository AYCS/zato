$ zato create-odb
$ zato drop-odb
$ zato create-lb-agent
$ zato create-server
$ zato quickstart

- create-odb - creates all SQL objects
- drop-odb - drops all SQL objects
- create-lb-agent - creates a load-balancer agent in a given directory
- create-server - creates a new server in a given directory
- quickstart - creates a new cluster and a server in a given directory;
  checks whether create-odb has been already called, creates the ODB if it hasn't.
